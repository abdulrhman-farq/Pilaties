# 🟦 المرحلة 1: ما قبل الافتتاح (اليوم -30 إلى 0)

### 1) تجهيز الموقع
- [ ] إنهاء عقد الإيجار واستكمال تراخيص البلدية ووزارة الرياضة والرخصة التجارية والسجل الضريبي (VAT).
- [ ] تنفيذ الديكور والتجهيز الداخلي (الأرضيات، الإضاءة، التكييف، المرايا).
- [ ] تركيب أجهزة Reformer الستة واختبار سلامتها + ترتيب الأدوات المساعدة (Props).
- [ ] تجهيز الاستقبال وغرف التبديل والخزائن ودورات المياه.

### 2) التوظيف والتدريب
- [ ] توظيف مدرّبتين معتمدتين + موظفة استقبال + عاملة نظافة (بدوام جزئي).
- [ ] تدريب الفريق على **منهجية موحّدة** للحصص ومعايير الجودة والسلامة.
- [ ] تدريب الاستقبال على نظام الحجز والتعامل مع العميلات وسياسات الإلغاء.

### 3) بناء الهوية والحسابات الرقمية
- [ ] إطلاق هوية بصرية كاملة (شعار، ألوان، قوالب سوشيال ميديا).
- [ ] إنشاء حسابات Instagram / TikTok / Snapchat / Google Business.
- [ ] جلسة تصوير احترافية للمكان والأجهزة والأجواء (**بنك محتوى لـ 60 يوماً**).
- [ ] تفعيل نظام الحجز والاشتراكات (Momence / Mindbody) مع الباقات والدفع الإلكتروني.

### 4) إطلاق حملة التسجيل المبكر (Early-bird)
- [ ] فتح **قائمة انتظار (Waitlist)** عبر صفحة هبوط بسيطة.
- [ ] إطلاق **عرض العضوية المؤسِّسة**: خصم 25% لأول 50 عميلة لمدة 3 أشهر.
- [ ] حملة تشويقية (عدّاد تنازلي + كواليس التجهيز) على Instagram/TikTok.
- [ ] فتح الحجز المسبق للحصص الأولى قبل الافتتاح.

# 🟩 المرحلة 2: الإطلاق (اليوم 1 إلى 30)

### 1) الافتتاح التجريبي (Soft Opening)
- دعوة 25–30 (مؤثرات + عميلات قائمة الانتظار) لحصص تجريبية مجانية.
- تصوير محتوى حقيقي (تجارب، انطباعات، أجواء) لإعادة استخدامه في الإعلانات.
- هدية افتتاح لكل حاضرة (Tote bag + كوبون إحالة).

### 2) عروض الاشتراك الأولى
| الباقة | السعر | السعر/حصة |
|---|---|---|
| حصة مفردة (Drop-in) | 150 ريال | 150 |
| باقة 4 حصص | 540 ريال | 135 |
| باقة 8 حصص | 1,000 ريال | 125 |
| باقة 12 حصة | 1,380 ريال | 115 |
| اشتراك شهري غير محدود (≈16 حصة) | 1,600 ريال | ≈100 |
- التركيز على تحويل العميلة من «حصة مفردة» إلى **باقة/اشتراك** لتأمين تدفق نقدي مقدّم وإشغال ثابت.

### 3) التعاون مع المؤثرين والشركاء المحليين
- التعاقد مع 3–5 مؤثرات محليات (Micro-influencers 10K–100K) شهرياً مقابل تجربة + رسوم رمزية.
- شراكات مع كيانات نسائية قريبة (صالونات، كافيهات صحية، عيادات تجميل) لتبادل العروض.
- إعلانات مدفوعة موجّهة جغرافياً (نطاق 5–8 كم) وعمرياً (إناث 25–45).

# 🟨 المرحلة 3: النمو والتثبيت (اليوم 31 إلى 90)

### 1) متابعة مؤشرات الأداء (KPIs)
- مراجعة أسبوعية للوحة المؤشرات (الاشتراكات، CAC، الاحتفاظ، الإشغال).
- تحديد **مصدر كل عميلة** لمعرفة أكفأ قناة اكتساب وإعادة توجيه الميزانية إليها.

### 2) تحسين نسب الإشغال
- إطلاق **تحدٍّ شهري** (12 حصة في 30 يوم → مكافأة) لرفع التردد.
- ملء فترات خارج الذروة بـ **حصص خاصة (Private/Duet)** بسعر 250–400 ريال.
- تعديل جدول الحصص حسب بيانات الطلب الفعلية (تعزيز الفترات الممتلئة).

### 3) حملات الاحتفاظ بالعميلات والإحالات
- **برنامج الإحالة:** «ادعي صديقة = حصة مجانية للاثنتين»؛ كل 3 إحالات = باقة 4 حصص.
- **برنامج ولاء بالنقاط:** كل 10 حصص = حصة مجانية.
- تواصل استباقي مع العميلات المتوقّع تسرّبهن (لم تحضر منذ أسبوعين) عبر عرض عودة.
- جمع شهادات فيديو واستطلاع رضا لتعزيز السمعة والتقييمات.

# 📊 لوحة مؤشرات الأداء (KPIs Dashboard)

| المؤشر | التعريف / طريقة القياس | الهدف |
|---|---|---|
| **عدد الاشتراكات الجديدة** | عميلات جديدة اشترين باقة/اشتراك شهرياً | 30–40 شهر 1 ← تراكم 80–100 |
| **تكلفة اكتساب العميلة (CAC)** | إنفاق الاكتساب ÷ عدد العميلات الجديدات | **≈ 350–450 ريال** |
| **القيمة الدائمة للعميلة (LTV)** | إيراد العميلة طوال فترة بقائها | **≈ 4,500–6,000 ريال** |
| **نسبة LTV / CAC** | كفاءة الاستثمار التسويقي (المعيار الصحي ≥ 3×) | **≈ 13–17×** ✅ |
| **معدل الاحتفاظ الشهري** | نسبة العميلات اللاتي يجددن/يستمرّن | **≥ 80%** (تسرّب ≤ 20%) |
| **معدل البقاء بعد 3 أشهر** | نسبة العميلات النشطات بعد 90 يوماً | **≥ 60%** |
| **نسبة الإشغال** | المقاعد المباعة ÷ الطاقة (1,092/شهر) | 20–30% ← **40–45%** |
| **معدل عدم الحضور (No-show)** | الحجوزات غير المحضورة | **≤ 10%** |

> **القراءة:** نسبة LTV/CAC المرتفعة جداً (≈ 13–17×) تعني أن كل ريال يُنفق على التسويق يعود أضعافاً — وهي من أقوى مؤشرات جدوى النموذج. التركيز التشغيلي يجب أن يكون على **رفع معدل الاحتفاظ** لأنه الرافعة الأكبر للربحية طويلة المدى.

## ✅ قائمة المتابعة الأسبوعية (للمالك/المدير)

- مراجعة لوحة KPIs (اشتراكات جديدة، CAC، احتفاظ، إشغال) واتخاذ قرار إعادة توزيع الميزانية.
- متابعة معدل عدم الحضور وتطبيق سياسة الإلغاء (قبل 12 ساعة).
- نشر 5–7 منشورات أسبوعياً والرد على كل الرسائل خلال ساعة.
- التواصل مع العميلات المتوقّع تسرّبهن قبل أن يغادرن.
- اجتماع أسبوعي قصير مع المدربات (جودة التجربة + ملاحظات العميلات).
