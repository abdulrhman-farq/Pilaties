# حزمة برومبتات احترافية — إنشاء ملفات مشروع استوديو البيلاتس

> انسخي **«بيانات المشروع»** أولاً، ثم ألصقيها قبل أي برومبت أدناه. كل برومبت مُصمَّم ليُنتج ملفاً احترافياً متّسقاً مع باقي الملفات.

---

## 🔒 بيانات المشروع (الصقيها قبل أي برومبت)

```
المشروع: استوديو Reformer Pilates نسائي بوتيكي — الرياض، السعودية.
المساحة: 120 م² (واجهة شارع 10م × عمق 12م) — وحدة زاوية بأول العمارة.
المدخل: من الفتحة الثانية (5م)، مع جدار خصوصية منحنٍ والاستقبال خلفه.
الأجهزة: 6 Reformer (صفّان × 3)، المسافة بين المحاور 1.4م، قابلة للتوسعة إلى 8.
+ ركن أجهزة كلاسيكية (Cadillac/Chair/Barrels) للجلسات الخاصة.
توزيع المساحة: استوديو 60م² · مدخل+استقبال+انتظار 21م² · تبديل+خزائن 13م² ·
  دورات مياه+دش 8م² · ركن أجهزة كلاسيكية 7م² · غرفة أدوات Props 6م² · مكتب المديرة 5م².
الفئة المستهدفة: سيدات 25–45، دخل متوسط إلى مرتفع. خصوصية نسائية تامة.
التسعير: سعر القائمة ≈172.5 ريال شامل الضريبة (الصافي 150) + باقات واشتراكات.
المالية: الاستثمار ≈275,000 ريال (يُوصى بتمويل 300,000) · تشغيل شهري ≈61,000 ريال ·
  الإيجار السنوي 204,000 · نقطة التعادل 37% · NPV@15% ≈653,000 · IRR ≈66% ·
  استرداد ≈13 شهر · أعمق قاع نقدي بالسنة 1 ≈ -106,000 (الشهر 5).
الهوية البصرية: كحلي #1F4E78 · تركوازي #2E8B8B · ذهبي #C9A227 · محايدات دافئة. هادئة وراقية.
الإضاءة: 2700K · CRI ≥ 95 · UGR < 16. ارتفاع السقف الصافي ≥ 3.0م.
اللغة: عربية (RTL) مع مصطلحات تقنية إنجليزية عند الحاجة.
```

## 1) دراسة الجدوى الاحترافية (تقرير/PDF)

```
أنت مستشار استثماري وخبير دراسات جدوى في قطاع البوتيك فيتنس بالسعودية.
اعتمد «بيانات المشروع» أعلاه حرفياً. أنشئ دراسة جدوى احترافية كاملة بتنسيق تقرير
رسمي (عناوين، جداول، رسوم) تتضمن الأقسام: ملخص تنفيذي · دراسة السوق (مع إطار
Trade Area) · الدراسة الفنية · التشغيلية · التسويقية · المالية (4 سيناريوهات
إشغال 30/50/70/90%) · المؤشرات (تعادل، ROI، NPV، IRR، استرداد) · منحنى التدفق
النقدي لسنة الإطلاق · تحليل الحساسية (السعر/الإيجار/الإشغال/الضريبة) · المخاطر ·
التوصيات. اشرح كل الافتراضات بوضوح، استخدم جداول مالية مرتّبة، وقدّم توصية
استثمارية نهائية كأنها مقدّمة لصندوق استثماري. اللغة عربية RTL، نبرة مهنية موجزة،
بألوان الهوية. أخرج المستند جاهزاً للطباعة A4.
```

## 2) النموذج المالي التفاعلي (Excel)

```
أنت محلل مالي. ابنِ نموذج Excel احترافياً متعدد الأوراق لمشروع «بيانات المشروع» أعلاه:
- ورقة «الافتراضات» (خلايا إدخال صفراء): الأجهزة، الحصص/اليوم، أيام التشغيل،
  السعر الصافي، الإيجار، بنود التكاليف، معدل الخصم.
- ورقة «التكلفة التأسيسية» وورقة «التكاليف التشغيلية».
- ورقة «سيناريوهات الإشغال» (30/50/70/90%) تحسب الإيراد والصافي وROI والاسترداد ونقطة التعادل.
- ورقة «توقعات 5 سنوات» مع NPV@12% وNPV@15% وIRR وتدفق تراكمي.
- ورقة «منحنى التدفق الشهري لسنة الإطلاق» (12 شهراً) تُظهر القاع وكفاية رأس المال العامل.
- ورقة «تحليل الحساسية» (السعر × الإشغال).
اربط كل المخرجات بصيغ حيّة من ورقة الافتراضات، نسّق بالألوان، وأضف رسوماً بيانية
(أعمدة للسيناريوهات، خط للتدفق التراكمي). تحقّق أن نقطة التعادل ≈37% وNPV@15%≈653,000
وIRR≈66% عند الإدخالات الأساسية.
```

## 3) عرض المستثمر (Pitch Deck — 12–14 شريحة)

```
أنت مصمم عروض استثمارية. صمّم Pitch Deck احترافياً (16:9، عربي RTL، ألوان الهوية)
لمشروع «بيانات المشروع» أعلاه، بالشرائح: الغلاف · الفرصة · المشكلة/الفجوة · الحل ·
السوق المستهدف · نموذج العمل (الباقات) · الاقتصاديات (تعادل 37%) · التوقعات
(سيناريوهات الإشغال) · مؤشرات العائد (NPV/IRR/ROI/الاسترداد) · خطة النمو ·
تحليل الحساسية · التدفق النقدي لسنة الإطلاق (رسم) · الطلب التمويلي (300,000 ريال) ·
لماذا الآن. كل شريحة: عنوان قوي + 3–5 نقاط أو جدول/رسم. نبرة واثقة موجزة. أخرجه
بصيغة PowerPoint قابلة للتعديل (نص حقيقي لا صور).
```

## 4) خطة التشغيل والتسويق — 90 يوماً

```
أنت مدير تشغيل وتسويق لاستوديوهات بوتيك. اكتب خطة تنفيذية لأول 90 يوماً لمشروع
«بيانات المشروع» أعلاه، بثلاث مراحل: (1) ما قبل الافتتاح — تجهيز الموقع، التوظيف
والتدريب، بناء الهوية والحسابات الرقمية، حملة التسجيل المبكر. (2) الإطلاق —
افتتاح تجريبي Soft Opening، عروض الاشتراك، التعاون مع المؤثرات والشركاء المحليين.
(3) النمو — متابعة KPIs، تحسين الإشغال، حملات الاحتفاظ والإحالات. أضف: ميزانية
تسويقية ≈40,000 ريال موزّعة على 3 أشهر (جدول)، ولوحة KPIs (الاشتراكات الجديدة،
CAC ≈350–450، LTV، نسبة LTV/CAC، معدل الاحتفاظ ≥80%، الإشغال، No-show ≤10%).
قوائم مهام قابلة للتحقيق، عربي RTL، بألوان الهوية.
```

## 5) مخطط الطابق المعماري (Floor Plan)

```
أنت مهندس معماري. ارسم مخطط طابق احترافياً مقياسياً (Scaled) لمشروع «بيانات المشروع»
أعلاه على مساحة 10م (واجهة) × 12م (عمق):
- الجدران الخارجية + واجهتان زجاجيتان معتمتان (أمامية + جانبية للزاوية).
- المدخل من الفتحة الثانية (5م) مع باب وحركة فتح.
- جدار خصوصية منحنٍ فور الدخول + مكتب الاستقبال خلفه + ركن انتظار.
- استوديو خلفي بـ 6 أجهزة Reformer (صفّان×3، مسافة محاور 1.4م، ممرات أمان).
- ركن أجهزة كلاسيكية (Cadillac/Chair) · تبديل+خزائن · دورات مياه+دش · غرفة Props · مكتب المديرة.
- أبعاد كل غرفة بالأمتار + جدول مساحات يُجمع على 120م² + سهم شمال + مقياس رسم.
ألوان الهوية، عناوين عربية RTL، إخراج A3 أفقي عالي الدقة.
```

## 6) لوحة RCP — مخطط السقف العاكس

```
أنت مهندس إضاءة/كهرباء. أنشئ لوحة RCP (Reflected Ceiling Plan) لمشروع «بيانات
المشروع» أعلاه. المبدأ الحاكم: العميلة مستلقية تنظر للأعلى → يُمنع وضع سبوتات
مباشرة فوق أجهزة الريفورمر؛ استخدم كوف محيطي + غسيل جدران غير مباشر. أظهر على السقف:
توزيع الإضاءة (2700K، CRI≥95، UGR<16، Dimmable؛ الاستوديو 150–250 lux، الاستقبال
300–500 lux/3000K، المرايا بلا ظلال)، الكوف المحيطي، حسّاسات الدخان والرشّاشات،
السمّاعات، منافذ التكييف الخطّية (بعيداً عن مسار الهواء على المتمرّنات)، فتحات
الصيانة، إضاءة الطوارئ ولافتات المخارج. مع مفتاح رموز (Legend) وجدول تجهيزات
الإضاءة. مقياسي، عربي RTL، A3 أفقي.
```

## 7) لوحة MEP / التكييف (مناخ الرياض)

```
أنت مهندس MEP. أنشئ لوحة تكييف وميكانيكا لمشروع «بيانات المشروع» أعلاه لمناخ الرياض
(صيف ~45°م، وحدة زاوية بواجهتين زجاجيتين = حمل تبريد مرتفع). أظهر: مواقع وحدات
التكييف (VRF/سبليت مقنّى) بمنطقتين (استوديو منفصل عن الخدمات)، مسارات الدكت ومنافذ
الهواء الخطّية المحيطية، الهواء الراجع، وحدة الهواء النقي (~140 L/s / ~300 CFM)،
شفّاطات الدش ودورات المياه، سخّان المياه، مسارات الصرف والعزل المائي. اذكر تقدير
حمل التبريد (~18–22 kW / 5–6 طن، يؤكّده الحساب التفصيلي)، والهدف الحراري 21–23°م،
وضجيج ≤ NC35. مفتاح رموز + جدول تجهيزات. عربي RTL، A3 أفقي.
```

## 8) لوحة الجدول الزمني والميزانية التقديرية

```
أنت مدير مشاريع. أنشئ لوحة «جدول زمني + ميزانية» لمشروع «بيانات المشروع» أعلاه:
- مخطط Gantt للمراحل: اختيار الموقع (30ي) · الترخيص عبر منصة «رياضي»/وزارة الرياضة
  (45ي متوازٍ) · التصميم والـBOQ (21ي) · التنفيذ/Fit-out (50ي) · توريد الأجهزة (30ي
  متوازٍ) · التوظيف والتدريب (30+20ي) · Soft launch ثم Full launch (14ي).
- جدول تفكيك الميزانية الرأسمالية: Fit-out 80,000 · أجهزة 41,000 · Props 15,000 ·
  لوحة وأنظمة 15,000 · رخص 15,000 · طوارئ 9,000 · رأس مال عامل 100,000 (يُوصى 120,000)
  = إجمالي 275,000 (تمويل موصى به 300,000).
- بند منفصل لخيارات الواجهة (F·01–F·06) لأنها قد تتجاوز سقف بند اللوحة الخارجية.
ألوان الهوية، عربي RTL، A3 أفقي.
```

```

الحالة المزاجية (هادئة، راقية، نسائية، حميمية)، لوحة الألوان (كحلي #1F4E78،
تركوازي #2E8B8B، ذهبي #C9A227، محايدات دافئة)، الخامات (مايكروسمنت للجدار المنحني،

لمسات بيوفيلية، نمط الإضاءة الدافئة. واكتب عبارة المفهوم: «6 أجهزة Reformer = سعة
الحصة الجماعية، مع أجهزة المنهج الكامل (Cadillac/Chair/Barrels) للجلسات الخاصة».
عربي RTL، A3.
```

## 10) لوحة خيارات الواجهة (Facade Options F·01–F·06)

```
أنت مصمم واجهات. اقترح 6 خيارات واجهة (F·01–F·06) للوحدة الزاوية في «بيانات المشروع»
أعلاه (واجهة شارع 10م = فتحتان×5م + واجهة جانبية). لكل خيار: لقطة تصوّرية، الخامات
(زجاج معتم/ستيل/ألمنيوم/إضاءة/لوحة اسم مضيئة)، مستوى الخصوصية النسائية، وتقدير تكلفة
نسبي (اقتصادي/متوسط/فاخر). راعِ الخصوصية التامة والهوية الراقية ولفت نظر المارّة دون
كشف الداخل. جدول مقارنة بين الخيارات. عربي RTL، ألوان الهوية، A3 أفقي.
```
