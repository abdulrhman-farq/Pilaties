# دراسة جدوى — استوديو Reformer Pilates نسائي | الرياض

دراسة جدوى اقتصادية احترافية وشاملة لمشروع استوديو **Reformer Pilates** نسائي في مدينة الرياض (100 م²، 6 أجهزة).

📄 **الوثيقة الكاملة:** [دراسة-الجدوى-استوديو-ريفورمر-بيلاتس-الرياض.md](./دراسة-الجدوى-استوديو-ريفورمر-بيلاتس-الرياض.md)

## محتويات الدراسة

1. دراسة السوق (حجم السوق، الطلب، المنافسون، الأسعار، الفجوات)
2. الدراسة الفنية (توزيع المساحة، الطاقة الاستيعابية، الحصص، الطاقم)
3. الدراسة التشغيلية (الهيكل الإداري، ساعات العمل، الحجز والاشتراكات)
4. الدراسة التسويقية (الإطلاق، Instagram/TikTok، المؤثرات، الإحالات)
5. الدراسة المالية (الاستثمار، التكاليف، الإيرادات، 4 سيناريوهات إشغال)
6. المؤشرات المالية (نقطة التعادل، ROI، IRR، NPV، الاسترداد)
7. تحليل المخاطر
8. التوصيات النهائية
