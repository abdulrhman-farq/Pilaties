## 📄 الوثائق

### 1. [الدراسة الكاملة](./docs/01-دراسة-الجدوى-الكاملة.md)
دراسة الجدوى الشاملة بثمانية أقسام: دراسة السوق، الدراسة الفنية، التشغيلية، التسويقية، المالية (4 سيناريوهات إشغال)، المؤشرات المالية، تحليل المخاطر، والتوصيات النهائية.

### 2. [خطة التشغيل والتسويق — 90 يوماً](./docs/02-خطة-التشغيل-والتسويق-90-يوم.md)
خطة تنفيذية بمراحل (ما قبل الافتتاح → الإطلاق → النمو) مع KPIs وميزانية تسويق وقنوات اكتساب مرتّبة حسب الكفاءة.

### 3. [عرض المستثمر](./docs/03-عرض-المستثمر-Pitch-Deck.md)
عرض تقديمي من 12 شريحة جاهز للعرض على مستثمر أو صندوق استثماري.

### 4. [النموذج المالي التفاعلي](./financial-model/)
ملف Excel بـ 5 أوراق: `الافتراضات` · `التكلفة التأسيسية` · `التكاليف التشغيلية` · `سيناريوهات الإشغال` · `توقعات 5 سنوات + المؤشرات (NPV/IRR)`. الأرقام **محسوبة ومخزّنة كقيَم** لتظهر بشكل صحيح على كل الأجهزة (الجوال والكمبيوتر).

> لتعديل الافتراضات وإعادة توليد الملف: عدّل قسم `INPUTS` في `build_model.py` ثم نفّذ:
> `pip install openpyxl && python3 financial-model/build_model.py`

### 5. ملفات العرض الجاهزة ([exports/](./exports/))
- **[دراسة-الجدوى-بيلاتس.pdf](./exports/دراسة-الجدوى-بيلاتس.pdf)** — نسخة PDF احترافية من الدراسة (عربي بمحاذاة RTL، 7 صفحات، جداول ملوّنة).
- **[عرض-المستثمر-بيلاتس.pptx](./exports/عرض-المستثمر-بيلاتس.pptx)** — عرض PowerPoint جاهز للعرض (13 شريحة).

> لإعادة التوليد: `pip install reportlab arabic-reshaper python-bidi python-pptx` ثم
> `python3 exports/build_pdf.py && python3 exports/build_pptx.py`

## 🎯 أبرز التوصيات

1. **المشروع مجدٍ اقتصادياً بدرجة عالية** — يكفي تجاوز 37% إشغال للربحية.
2. **البدء بـ 6 أجهزة** ثم إضافة جهازين عند تجاوز 70% إشغال (المساحة تستوعب 8 بنفس الإيجار).
3. **التسعير:** 150 ريال للحصة المفردة كنقطة دخول + باقات واشتراكات لتعظيم الولاء والتدفق النقدي.
4. **عامل النجاح الحاسم:** تسويق قوي عند الإطلاق + علامة لا تعتمد على مدربة بعينها.

---

> **إخلاء مسؤولية:** الأرقام مبنية على افتراضات موضّحة صراحةً في الدراسة، وتتطلب تحديثاً عند تثبيت العقود الفعلية (الإيجار، فاتورة الأجهزة، الرواتب). يُنصح بمراجعة محاسب قانوني ومستشار ضريبي قبل التنفيذ.
